C. Division

Time Limit: 1 seconds

Memory Limit: 512 MB

Problem Statement

Oleg's favorite subjects are History and Math, and his favorite branch of mathematics is division. To improve his division skills, Oleg came up with \(t\) pairs of integers \(p_{i}\) and \(q_{i}\) and for each pair decided to find the greatest integer \(x_{i}\), such that:

  • \(p_{i}\) is divisible by \(x_{i}\)
  • \(x_{i}\) is not divisible by \(q_{i}\).
Oleg is really good at division and managed to find all the answers quickly, how about you?

Input

Input The first line contains an integer \(t ( 1\leq t \leq 50 ) \) — the number of pairs. Each of the following \(t \) lines contains two integers \(p_{i}\) and \(q_{i}\) \( (1 \leq p_{i} \leq 10^{18} )\);\( (2 \leq q_{i} \leq 10^{9} )\) — the \(i \)-th pair of integers.

Output

Print \(t\) integers: the \(i\)-th integer is the largest \(x_{i}\) such that \(p_{i}\) is divisible by \(x_{i}\), but \(x_{i}\) is not divisible by \(q_i\). One can show that there is always at least one value of \(x_{i}\) satisfying the divisibility conditions for the given constraints.

Example

Input:
5
1 2 3 4 5

Output:
15
